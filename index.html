Features

Everything your PDFs need.

Nine core surfaces, 46+ tools beneath them. Every action runs on-device — no uploads, no accounts, no background calls home.

⌘K

Library & Organisation

Browse folders in list or grid view. Sort, filter, favourite, and full-text search across your whole library. Command Palette (⌘K) jumps to any tool.

⌘⇧E

Metadata Management

Batch edit titles, authors, subjects, and dates. Rename with templates, copy metadata between files, apply folder-wide presets, and validate integrity.

⌘⇧K

Compress

Six PDFium-powered presets — from Lossless stream optimisation to Maximum 72 DPI crunch. CMYK preserved on Minimal and High Quality for print workflows.

⌘⇧M

Page Manager

Reorder, rotate, duplicate, extract, and insert blanks. Merge files, split on any page range, crop, and normalise page sizes. Non-destructive — originals are never overwritten.

⌘⇧P

Password & Watermark

Protect or unlock files with a password. Apply watermarks, headers, footers, stamps, and backgrounds — to one file or an entire batch in a single step.

Analysis & Validation

Find duplicates, generate library statistics, check PDF/A compliance, run validation reports, and export metadata to CSV or JSON.

OCR

Apple Vision on-device OCR. 20+ languages. Configurable DPI, auto-rotate, deskew. Export text to sidecar .txt or clipboard. Live per-page progress, cancellable at any moment.

Redact & Privacy

Permanently black-box matching text — burned into page content, not recoverable. Match Preview counts hits per file before you commit. Safe for client work and legal exports.

Forms & Signatures

Fill interactive PDF forms: text fields, checkboxes, radio groups, dropdowns. Draw, type, or upload an electronic signature and save it for reuse across documents.

FAQ

Common questions.

Is my data sent anywhere?
No. All PDF processing happens locally on your Mac or iPad using Apple's native frameworks. File contents, metadata, and usage data never leave your device. The only outbound requests are optional Sparkle update checks and one-time license activation.
Does it work on Apple Silicon? And iPad?
Yes to both. PDFloki ships as a universal binary — fully native on Apple Silicon (M1 and later) and Intel Macs, plus a native iPadOS 17+ build designed for touch.
What's the difference between compression presets?
Six levels from Lossless (2–10% savings, stream optimization only) to Maximum (60–80% savings, 72 DPI). Levels 1–2 preserve CMYK color for print workflows. Level 3 Standard is the recommended default for most day-to-day use.
Do I need an internet connection?
No — PDF processing works fully offline. The only network requests are optional update checks via Sparkle and the one-time license activation. Everything else is local.
What happens after the 7-day trial?
After the trial, a subset of tools remains free. A one-time purchase unlocks every feature permanently — no recurring fees, no seat counts, no surprises.
